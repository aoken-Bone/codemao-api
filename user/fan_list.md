# 粉丝列表

> 获取任意用户的粉丝  
> `/creation-tools/v1/user/fans` `GET`

## URL 参数

|   key   |    value     | type |        |
| :-----: | :----------: | :--: | :----: |
| user_id |   用户 ID    | int  | `必须` |
| offset  |     页码     | int  | `必须` |
|  limit  | 每一页的数量 | int  | `必须` |

## 返回内容

> 与[关注列表](/user/follow_list.md)完全一样
